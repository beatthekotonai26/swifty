Custom Cursor Files

This directory should contain the following cursor image files:
- gaming.png - A gaming-style cursor for normal state
- gaming-pointer.png - A gaming-style cursor for pointer/clickable state
- retro.png - A retro-style cursor for normal state
- retro-pointer.png - A retro-style cursor for pointer/clickable state

You can find free cursor images at websites like:
- https://www.cursor.cc/
- https://custom-cursor.com/
- https://www.flaticon.com/

The recommended size for cursor images is 32x32 pixels.
Make sure to use PNG files with transparency for the best results.
